### node

* 本插件无需安装, 在安装 `nodejs` 时会一并安装

* 执行脚本文件
    * `node 脚本文件.js` 

* 全局变量
    * `__dirname` 文件夹路径
    * `__filename` 文件路劲
    * `global` 全局资源

* 包含模块
    * `events` 事件模块
    * `fs` 文件模块
    * `http` 网络服务模块
    * `request` 发起请求
    * `child_process` 多线程
    * `os` 系统
    * `path` 路径
    * `util` 工具包
    * `co` 生成迭代器
    * `node-fetch` node 使用的 fetch
